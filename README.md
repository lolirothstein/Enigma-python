A Python implementation of the classic Enigma game.

The objective is to guess a randomly generated five-digit secret number. After each attempt, the game provides feedback indicating how many digits are correctly positioned and how many are present in the secret number but placed in the wrong position.

This project was developed as a way to practice Python programming, with a focus on writing modular, readable, and reusable code.

Features
🎲 Randomly generated five-digit secret number.
🔢 Supports repeated digits.
✅ Input validation.
🎯 Choice between limited or unlimited attempts.
🏳️ Option to surrender when playing with unlimited attempts.
🔁 Play multiple games without restarting the program.

How to Play:
Run the script.
Choose whether you want to play with a limited number of attempts.
Enter a five-digit number.
After each guess, you'll receive feedback:
Correctly positioned digits: correct digit in the correct position.
Misplaced digits: correct digit in the wrong position.
Keep guessing until you find the secret number or run out of attempts.

Technologies:
Python 3
Standard Library (random)
